# 拼图生成器

一个浏览器端的拼图生成工具，可以选择精美的布局模板，自动将图片填入对应位置，支持手动调整，导出为 PNG 图片。

## 功能特性

### 模板系统
- **18 款精美模板**，分为 5 类：
  - 网格系：2x2、3x3、2x3
  - 杂志风：1大2小、1大4小、横版大片、英雄照
  - 瀑布流：Pinterest 风格错落布局
  - 边框系：带边距的卡片式布局
  - 创意系：马赛克、楼梯、错位重叠、全景三联

### 核心功能
- 模板选择预览
- 多图片选择（支持一次选择多张）
- 自动填充（Cover 模式，等比裁剪填满区域）
- 手动调整：
  - 拖拽移动图片位置
  - 滚轮缩放图片
  - 触摸屏双指缩放
- 自定义输出尺寸
- 导出 PNG

### 隐私安全
- **纯前端运行**，所有图片在本地处理
- 不上传到任何服务器

## 使用流程

1. **选择模板**：点击左侧模板卡片切换布局
2. **添加图片**：点击"+ 选择图片"按钮，从文件夹选择多张图片
3. **调整图片**：
   - 鼠标悬停查看槽位
   - 拖拽移动图片位置
   - 滚轮缩放图片大小
4. **设置尺寸**：根据需要调整输出宽度和高度
5. **导出**：点击"导出 PNG"按钮保存拼图

## 技术栈

- HTML5 + CSS3 + JavaScript (ES6+)
- Canvas API 图片处理
- File API 文件处理
- 无需任何后端服务
